## Color Area

A color area allows users to adjust two channels of an RGB, HSL or HSB color value against a two-dimensional gradient background.

<Playground comp="colors/color-area" />

## Color Swatch

A ColorSwatch displays a small preview of a selected color, helping users see the exact shade they've chosen at a glance.

<Playground comp="colors/color-swatch" />

## Color Slider

A color slider allows users to adjust one individual channel of a color value, such as red, green, or blue, giving precise control over the color’s intensity.

<Playground comp="colors/color-slider" />

## Color Swatch Picker

A ColorSwatchPicker displays a list of color swatches, allowing users to easily browse and select one color from the available options.

<Playground comp="colors/color-swatch-picker" />

## Color Input

A color input allows users to edit a hex color code or adjust individual color channel values, providing flexible control over color selection.

<Playground comp="colors/color-input" />

## Color Picker

A ColorPicker synchronizes a color value between multiple React Aria color components. It simplifies building color pickers with customizable layouts via composition.

<Playground comp="colors/color-picker" />
