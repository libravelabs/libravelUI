Add `cn` helper using `clsx` and `twMerge` for conditional class merging in Tailwind components.

```ts ts2js title="lib/utils.ts"
import { clsx, type ClassValue } from "clsx";
import { twMerge } from "tailwind-merge";

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}

```

```tsx
  name="username"
  type="text"
  label="Username"
  description="Enter your username"
  placeholder="ihatejunk_"
  errorMessage="This field is required"
  startContent={"..."}
  endContent={"..."}
/>
```

## Examples

### Sizes
Defines the height and overall spacing of the input. Use different sizes to match the component to various UI contexts.
<SizesTextField />

### Variants
Controls the visual style of the input, such as default or error states. Use variants to reflect the input’s purpose or status.
<VariantsTextField />

### Start and End Content
This example shows how to add customizable elements at the start and end of the input field, such as icons or buttons, to enhance user interaction and improve UI clarity.
<StartEndContentTextField />

This example demonstrates how to integrate the text field component within a form context, showcasing form validation, error handling, and submission behavior seamlessly.
